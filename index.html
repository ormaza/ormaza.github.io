11.Canny e a arte com pontilhismo

cannypoints.cpp

Nesse projeto foi pedido para se gerar uma imagem pontilhista usando as bordas produzidas pelo algoritmo de Canny para melhorar a qualidade dessa técnica. Assim primeiramente foi aplicada a técnica pontilhista em toda a imagem, usando circulos de tamanho grande para essa parte.

Após isso criei um laço de repetição que faz variar o limiar utilizado no filtro de canny. Nesse filtro quanto menor o limiar mais bordas são capturadas. Internamente no laço, em cada borda capturada serão desenhados pontos de raio randomico, mas com tamanho máximo bem inferior ao passo anterior apresentado. Esses circulos tem planos de fundo que coincidem com a imagem original.
