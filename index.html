4.Preenchendo regiões

labeling.cpp

Foi pedido uma solução para o problema de existir mais de 255 objetos na cena. Isso gera problema pois a imagem que temos em tom de cinza varia seus niveis de 0 á 255. Como o floodfill está marcando uma dada região com o contador de objetos esse valor no caso acima irá ultrapassar 255. A solução que achei foi usar a função min() da biblioteca math retornando como valor máximo para o floodfill o valor 255. min(255,nojects) significa que se nojects>255 retorne 255 senao retorne nobjects.
